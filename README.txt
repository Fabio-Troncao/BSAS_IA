A classe representa o algoritmo de agrupamento BSAS - esquema algorítmico sequencial básico.

O algoritmo tem dois parâmetros obrigatórios: o número máximo permitido de clusters e o limite 
de dissimilaridade ou, em outras palavras, a distância máxima entre os pontos. A métrica de 
distância também pode ser especificada usando parâmetros 'métricos', por padrão, a distância 
'Manhattan' é usada.

Os resultados de agrupamento desse algoritmo dependem da ordem dos objetos nos dados de entrada.

Para usar o pyclustering:

pip3 instalar pyclustering

Para usar o sklearn.datasets:

pip install scikit-learn
